# 🎯 AI Lotto System

1~45 범위에서 중복되지 않는 번호 6개를 랜덤으로 선택해
자동 추천 번호 10줄을 생성하도록 구현했습니다.

랜덤 생성과 데이터 기반 분석을 함께 활용하여
각 알고리즘별 특징을 반영한 번호 생성 구조를 구현했습니다.

---

## 📌 주요 기능

- 📊 과거 로또 데이터 기반 분석
- 🤖 AI 모델별 번호 생성 방식 제공
  - Random Forest
  - RNN
  - XGBoost
  - TensorFlow
- 🎯 알고리즘별 결과 비교 UI
- 🔄 랜덤 + 확률 기반 혼합 생성
- 📦 JSON 데이터 기반 로컬 분석

---

### 🤖 모델별 번호 생성 로직 (프로젝트 적용 방식)

각 알고리즘은 실제 학습 모델이 아닌,
로또 데이터의 특성에 맞게 **로직 형태로 단순화하여 적용**했습니다.

---

#### 🌲 Random Forest 방식

번호를 하나의 기준이 아니라
여러 조건을 동시에 고려하여 선택하는 구조로 구현했습니다.

* 번호별 출현 빈도 계산
* 자주 함께 등장한 번호 조합(Pair) 반영
* 조건별 점수를 분리해서 계산 후 통합

👉 즉, 하나의 기준이 아닌
여러 기준을 종합해서 번호를 선택하는 방식입니다.

---

#### 🔁 RNN 방식

회차 흐름을 기반으로
최근 데이터의 영향을 더 크게 반영하도록 설계했습니다.

* 최근 회차 데이터에 높은 가중치 부여
* 이전 회차와 겹치는 번호 확률 증가
* 연속 출현 패턴 반영

👉 즉, 과거 전체가 아닌
“최근 흐름”을 따라가는 번호 생성 방식입니다.

---

#### ⚡ XGBoost 방식

번호별 중요도를 계산하고
가중치를 기반으로 선택 확률을 조정하는 구조입니다.

* 번호별 등장 빈도 → 가중치 변환
* 특정 조합 반복 여부 반영
* 높은 점수를 가진 번호가 선택될 확률 증가

👉 즉,
“잘 나오는 번호를 더 잘 나오게 만드는 구조”입니다.

---

#### 🧠 TensorFlow 방식

여러 요소를 동시에 고려하여
확률적으로 번호를 선택하는 방식으로 구현했습니다.

* 빈도 + 조합 + 랜덤 요소를 함께 반영
* 확률 분포 기반으로 번호 선택
* 완전 고정되지 않은 유동적인 결과 생성

👉 즉,
고정된 규칙이 아닌 확률 기반으로 균형 있게 번호를 생성합니다.

---
